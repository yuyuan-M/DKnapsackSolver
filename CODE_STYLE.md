# 代码规范说明

本项目遵循 **PEP 8** 编码规范，并补充以下细则：

- **缩进**：使用 4 个空格，禁止使用 Tab。
- **行宽**：每行最多 79 个字符（特殊情况放宽至 99）。
- **命名**：
  - 类名：`UpperCamelCase`
  - 函数/方法名：`lower_case_with_underscores`
  - 变量名：`lower_case`
  - 常量：`UPPER_CASE`
- **空行**：
  - 类之间空两行
  - 函数之间空一行
  - 类内方法之间空一行
- **注释**：
  - 所有公共类、函数使用 docstring 说明
  - 复杂逻辑添加行内注释
- **操作符空格**：二元操作符两侧加空格，逗号后加空格

具体细则参照 [PEP 8](https://www.python.org/dev/peps/pep-0008/)。
